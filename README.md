### Data Set Information:
The Covid patient datasets dataset is collected by randomly sampling

- age
- Sleep_problems
- Headache
- Diarrhea
- Abdominal_pain
- body_pain
- Body_discoloration
- Cough
- Fever
- Ague
- Sore_throat
- Fatigue
- runny_nose
- Chest_pain
- Decreased_appetite
- Vomit
- Nausea
- Sneezing
- Shortness_of_breath
- Loss_of_smell
- Loss_of_taste
- urticaria

values from covid patient symptoms of various age groups, race groups, and genders obtained from semnan University. Total sample size is 417.
